#**Fin_Tracker**

Fin_tracker is a system or tool used for 'monitoring' and 'recording' expenses and income for financial awareness management.


##**Features**

1. **User Auth🔐**: Stores users data from sign_up for Login and managing users activities..

2. **Income, Expence and Available Income Recording**: Provides a section for recording user's data (expenses, income and Available income)

3. **Activity History**: Provides a section for user to view past record's of incomes and expenses with the date and time provided..

4. **Dashboard**: provides a dashboard where the user can view total expenses and total income..

5. **Data persistence**: Manages user's data for login and users activities (history, total income and expenses).


##**How To Use 📕**

Run the code by entering "python FintrackerAcc.py"

1. **Account creation**: select (1) to sign-up or setup an account once done you will be taken to the login section .

2. **Login**: On the login section input your details correctly and you will be taken to the 'Home menu' you've logged-in successfully..

3. **Navigation**: 

   select (1) to view dashboard and see your progress (total income, total expenses and Available income)
   select (2 / 3) to record incomes or expenses 2-for income 3-for expenses 
   select (4) to view Activity History to view record of incomes and expenses you have made..
   select (5) to log-out you will be taken to the main menu...
   select (6) to exit the program..

easy and interesting to navigate 


## **Project Structure ⚙️**

### FintrackerAcc.py: 
This manages the user auth where the user sign-up or log-in

### FintrackerMain.py: 
This manages the main program where the user record's income and expenses, activity history and dashboard...

### Fintracker_acc.json: 
This stores the user's login details (username and password) for login purpose..

### FintrackerMain.json: 
This stores the users activities and data (Activity history, total income, total expenses and Available income) for data retrieval when user log-in..
